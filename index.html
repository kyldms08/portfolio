BrightStar Hybrid Network Infrastructure — BSCNE Capstone
BSCNE Capstone · AWS Site-to-Site VPN · GNS3 · 8 Test Cases · WGU
✓ All 8 Tests PassedCapstone · GradedAWS + GNS3 + VyOS

Designed and deployed a fully functional hybrid network infrastructure connecting an on-premises GNS3 environment to AWS via a site-to-site IPSec VPN. The on-premises side uses VyOS as a router, DHCP server, and zone-based firewall with 802.1Q VLAN trunking segmenting Staff (VLAN 10) and Management (VLAN 20) traffic. The AWS side consists of a custom VPC with public and private subnets, Internet Gateway, NAT Gateway, Virtual Private Gateway, Security Groups, and NACLs. Eight test cases validate every layer of the architecture — from Layer 2 VLAN segmentation to end-to-end encrypted VPN connectivity and defense-in-depth security enforcement at both the on-premises and cloud layers.
